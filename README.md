# Remix Solidity 示例

Solidity 智能合约学习与示例项目。

## 参考资源

- **[Solidity by Example](https://web3dao-cn.github.io/solidity-example/)** — Solidity 中文示例教程

## 项目结构

```
src/
├── 01-20  基础语法（变量、函数、循环、数组等）
├── 21-30  进阶概念（映射、结构体、继承、事件等）
├── 31-40  高级特性（多继承、可见性、接口、调用等）
├── 41-54  实战示例（ERC20、多签钱包、荷兰拍卖等）
└── SOL/   其他 Solidity 示例
```

## 运行

使用 [Remix IDE](https://remix.ethereum.org/) 或本地 Hardhat/Foundry 环境编译与部署。
